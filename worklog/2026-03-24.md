## 이슈 및 해결
- **Skia API 변경**: `useTouchHandler` 제거됨 → `react-native-gesture-handler`의 `Gesture.Pan()` 사용으로 전환
- **expo-file-system API 변경**: `documentDirectory`, `EncodingType` 등 제거 → 클래스 기반 `File`, `Directory`, `Paths` API로 전환
- **Node.js 버전 경고**: v20.18.0 사용 중, RN 0.83이 >=20.19.4 요구. 현재까지 동작에는 문제 없으나 업데이트 권장
