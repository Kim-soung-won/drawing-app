# Pub Diary - 어플리케이션 기획서

---

## 1. 개요

**Pub Diary**는 손글씨로 작성한 사색, 일기, 에세이 등을 저장하고 공유할 수 있는 모바일 어플리케이션입니다.

AI와 디지털화가 가속되는 시대에, 인간만의 감성을 담은 손글씨의 가치를 재발견하고 이를 통해 사람들이 서로의 일상과 사색을 자연스럽게 나눌 수 있는 공간을 지향합니다.

---

## 2. 기획 의도

| 항목 | 내용 |
|------|------|
| **목적** | 손글씨로 작성한 사색, 일기, 에세이 등을 저장하고 공유 |
| **근거** | 손글씨를 활용한 커뮤니티의 부재. 손글씨만의 매력은 온라인 텍스트와는 다른 감성을 전달하며, 이를 취미로 즐기는 사람들이 존재 |

---

## 3. 핵심 가치

- **진솔함** — 손글씨 특유의 꾸밈없는 솔직함
- **유니크함** — 사람마다 다른 필체가 만드는 개성
- **따뜻함** — 디지털 흑백에서 벗어난 인간적인 감성
- **연결** — 나만의 글을 세상과 자연스럽게 나누는 경험

---

## 4. 주요 기능

### 4.1 글쓰기 (Core)

- 태블릿 + 스타일러스펜 환경에서의 손글씨 작성
- 단순하고 직관적인 글쓰기 인터페이스 (참고: **Goodnotes**)
- 펜 종류, 굵기, 색상 등 기본적인 필기 도구 제공
- 작성한 글의 저장 및 관리 (일기장/노트북 단위)

### 4.2 OCR 지원

- 손글씨 텍스트 인식을 통한 가독성 점검
- 필체 분석 및 피드백
- 검색 기능 지원 (손글씨 내용을 텍스트로 변환하여 검색 가능)

### 4.3 공유 & 커뮤니티

- 개인이 제작한 손글씨 컨텐츠를 선택적으로 공유
- 다른 사람의 사색과 일상을 자연스럽게 탐색
- 공유된 글에 대한 감성적 반응 (좋아요, 공감 등)

### 4.4 개인 보관

- 나만의 일기장으로 비공개 사용 가능
- 날짜별, 태그별 정리 및 아카이빙
- 작성 히스토리 및 통계

## 5. UI/UX 방향성

### 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **서정적이고 따뜻한 분위기** | 손글씨의 감성에 어울리는 톤앤매너. 부드러운 색감과 여백 활용 |
| **혼자 사용하는 감성** | 조용한 방에서 노트를 펼치는 듯한 몰입감. 과도한 소셜 요소 배제 |
| **단순한 글쓰기 환경** | 기술적 제약보다 글쓰기 경험 자체에 집중. Goodnotes 스타일의 직관적 인터페이스 |
| **손글씨의 개성 존중** | 필체의 유니크함이 드러나는 레이아웃. 획일화된 폰트가 아닌 사용자 고유의 글씨가 주인공 |

## 6. 논의 사항

### 6.1 입력 방식

> 필수 준비물이 태블릿과 S펜이 되어야 할까?

- 태블릿 + 스타일러스: 가장 자연스러운 필기 경험 제공
- 종이 노트 촬영 후 이미지 업로드: 촬영 환경(조명, 각도, 그림자 등)에 따라 품질 차이가 크고 고려할 요소가 많음
- **우선 접근**: 태블릿 기반 필기를 1차 지원, 이미지 업로드는 추후 검토

### 6.2 가독성

> 남의 손글씨는 알아보고 읽기 어렵다.

- OCR 기반 텍스트 변환을 함께 제공하여 가독성 보완
- 원본 손글씨 + 변환 텍스트를 병렬로 보여주는 방식 검토
- 필체 가독성 점수 등을 통해 작성자에게 피드백 제공

### 6.3 사운드

> 사운드가 있다면 그 출처는?

- 글쓰기 몰입을 돕는 배경 사운드(빗소리, 연필 소리 등) 검토
- 저작권 문제 해결 필요 (무료 음원 라이브러리, 자체 제작 등)
- 1차 버전에서는 사운드 기능 제외 후 추후 검토

### 6.4 개발 우선순위

> 기술적 제약은 뒤로 미룬다. 단순한 글쓰기 환경을 UI로 우선 구현해본다.

- **Phase 1**: 단순한 글쓰기 환경 UI 구현 (태블릿 필기)
- **Phase 2**: 저장/관리 기능
- **Phase 3**: OCR 및 가독성 점검
- **Phase 4**: 공유 및 커뮤니티 기능

## 7. 타겟 사용자

- 손글씨 쓰기를 취미로 즐기는 사람
- 디지털 일기를 쓰되, 텍스트 입력이 아닌 필기 경험을 원하는 사람
- 자신의 사색과 감성을 조용히 기록하고 싶은 사람
- 다른 사람의 손글씨 일기를 통해 일상의 위로를 받고 싶은 사람
